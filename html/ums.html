광주은행 UMS 솔루션 구축

  • HTML
  • CSS
  • jQuery
  • JAVA

UMS는 관리자를 위한 통합 메시징 시스템으로 이메일발송, 실시간 자동응답 서비스, SMS, 모바일 푸시 알림 등 총 4가지 서비스로 구성되어 있는 솔루션입니다. 은행을 대상으로 한 프로젝트인 만큼 웹표준과 IE 등의 브라우저 및 버전 호환성 등에 특히 신경쓰며 마크업 하였습니다. 구축부터 운영까지 염두에 두었고, 예외 케이스를 고려하여 어떤 상황에서도 틀어짐이 적도록 UI를 개발하였습니다.

기획자, PM과의 활발한 커뮤니케이션을 통해 즉각적으로 요구사항을 파악하여 애자일 방식에 나름의 질서를 더해 업무를 수행하였습니다. 또한 2인 체제로 협업하며 서로 고민되는 부분에 대해 지식을 공유하였기에, 보다 일관성있고 경제성 있는 코드를 작성할 수 있었습니다.

그래프를 활용한 대시보드 개발을 위해 ECHARTS 플러그인을 활용하였습니다. 공통요소와 서비스별 예외요소를 나누는데 중점을 두어 CSS를 분리하였고, 레이아웃에 대해서는 보수적으로 접근하되 레거시를 지양하며 코드를 작성하였습니다. 페이지마다 스크롤양을 최소화하였기 때문에 각 요소에 scroll-bar를 형성해야 했습니다. 브라우저 별로 호환되도록 설정하였고 스크롤 생성 시 레이아웃에 위화감이 없도록 대응하여 완성도를 높였습니다.
