이젠클라우드 BtoC 쇼핑몰 서비스 구축

  • Axure
  • HTML
  • CSS
  • jQuery
  • Java

이젠클라우드는 BtoC 프로젝트로 쇼핑몰 관리 서비스를 제공하는 솔루션입니다. JSP 환경에서 UI개발을 하였고 관리자서비스(어드민)부터 쇼핑몰, 마이오피스(커스토머 어드민)까지 솔루션 전체의 퍼블리싱을 담당하였습니다. BE와 주도적으로 대화하며 마크업과 기획의도를 설명하며 보다 명확하게 협업하고, STS와 git bash를 활용하여 업무를 공유하였습니다.

초반 쇼핑몰 기획의 40%를 담당하였고, 선임 기획자와 함께 논의한 결과 복잡한 다단계 로직을 효율적으로 정돈할 수 있었습니다. 처음 맡아보는 업무였기에 문서작성과 디자인 검수 등의 어려움을 겪었지만 서비스 전체의 완성을 바라보며 최선을 다 한 프로젝트입니다.

소개사이트 방문하기

비즈니스 플로우는 임시 데이터와 이미지를 넣고 프로토타입을 만든 뒤 고객사에 제안하는 형태였습니다. 쇼핑몰 파트의 경우 프로토타입 안에서도 다단계 산업의 특성을 조화롭게 채운 서비스를 만들기 위해 팀원들과 함께 고뇌하며 UI를 만들었어요.

다단계 쇼핑과 비즈니스가 80% 이상 모바일을 통해 이루어지기 때문에 어떤 기기에서도 사용하기 편하도록 반응형으로 작업했습니다.

어드민의 경우 매출관리(전사업무, 회원, 수당 등)부터 쇼핑몰 관리까지 지원합니다. 단순한 디자인으로 템플릿을 만들어 컴포넌트의 재사용성을 높이는데 중점을 두고 마크업 하였습니다. 해당 템플릿을 바탕으로 BE가 개발 업무를 좀더 빠르게 처리할 수 있어요.
